翻译软件：


纯文本
```
纯文本浏览器里翻译：比如 沉浸式翻译 等（会有原文和翻译的文字），而且iPhone的Safari里也可以使用，这个软件还有一些有意思的地方：
    可以鼠标悬停到文字上面就会自动翻译，如果觉的没必要，可以悬停然后按下ctrl也能翻译。
    而且文本框中输入中文，按3个空格，能自动变成英文。
    沉浸式翻译 还一个很牛逼的地方就是，在油管里，字幕都是英文的，能自动翻译。
一般都是浏览器插件调用了第三方搜索服务实现的。
```


图片
```
图片翻译：
1. 快速的可以微信截图，里面可以翻译  
2. 比如使用其它第三方软件，有道翻译等。
如果想知道更多截图翻译软件，可以搜索 OCR翻译软件，会出来很多，OCR是视觉字符识别的意思。
```


文件
```
整个pdf翻译：可以Google translate里上传文件，直接翻译了。
```


phpstorm中安装翻译软件
```
在phpstorm的plugin中搜索translate这个插件，安装上就行
修改快捷键：File -> Settings 在设置窗口中，导航到 Keymap
修改翻译引擎： File -> Settings (Windows/Linux) 或 PhpStorm -> Preferences (macOS)，导航到 Tools -> Translation  可以修改翻译引擎
```
