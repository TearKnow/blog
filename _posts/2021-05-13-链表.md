### 1.2.迭代-反转一个链表
```php
function reverse($head){
    $prev = null;
    $curr = $head;
    while($curr){
        $tmp = $curr->next;//缓存，下一个要将是curr

        $curr->next = $prev;//当前的下一个是上一个
        $prev = $curr;//记录当前，作为下一个的prev

        $curr = $tmp;//获得缓存
    }
    return $prev;
}
```
