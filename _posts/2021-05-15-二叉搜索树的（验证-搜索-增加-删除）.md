### 二叉搜索树中的搜索(Leetcode700)
```php
function searchBST($root, $val) {
    if(!$root){
        return null;
    }
    if($root->val == $val){
        return $root;
    }
    if($root->val > $val){
        return $this->searchBST($root->left, $val);
    }
    if($root->val < $val){
        return $this->searchBST($root->right, $val);
    }
}
```
