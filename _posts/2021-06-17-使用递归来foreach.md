一般都是foreach来进行数组的遍历，这里使用递归来实现
```php
<?php
    function foreachGetNums($nums, $index){
        if($index >= count($nums)){
            return false;
        }
        echo $nums[$index]."\n";//这里是正的打印1,2,3,4,5
        foreachGetNums($nums, $index + 1);
        //如果在这里，就是反的打印, 5,4,3,2,1
    }

    foreachGetNums([1, 2, 3, 4, 5], 0);
```
